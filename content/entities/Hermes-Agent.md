# Hermes Agent

Agente autónomo automejorable con capacidad de aprendizaje a largo plazo.

- **Característica clave:** Creación de habilidades desde la experiencia.
- **Plataforma:** Agnóstico (soporta Linux, macOS, WSL2, Windows, Android).
- **Setup:** `hermes setup --portal` para acceso unificado a herramientas.

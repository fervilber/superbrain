# Actualización Técnica: Persistencia de Agentes y Memoria en Hermes

Hoy el sistema se centra en la arquitectura de memoria persistente de **Hermes Agent**. A diferencia de otros agentes, Hermes no solo utiliza un historial de chat, sino un sistema de **memoria estructurada** que sobrevive a las sesiones.

## Características de la Memoria

1. **Memoria Persistente**: Hechos almacenados que se inyectan en futuros turnos para reducir la necesidad de recordatorios.
2. **Contexto Dinámico**: La capacidad del agente para discernir qué información es estable (preferencias, conventions) vs temporal (progreso de tareas).
3. **Aprendizaje de Experiencias**: El sistema utiliza la gestión de "Skills" para procedimientos recurrentes, lo que permite que el agente "aprenda" cómo resolver problemas complejos de forma autónoma con el tiempo.

Este enfoque permite a Hermes construir un modelo de usuario cada vez más profundo, optimizando los flujos de trabajo sin intervención manual.

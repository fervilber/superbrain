# Subagent Delegation en Hermes Agent

El ecosistema de Hermes Agent destaca por su capacidad de **delegación de subagentes** mediante la herramienta `delegate_task`. Esta funcionalidad permite gestionar tareas complejas de manera eficiente.

## Características Clave
- **Aislamiento:** Cada subagente nace con una conversación nueva, sin conocer el historial del agente padre.
- **Contexto explícito:** El agente padre debe inyectar todo el contexto necesario (código, errores, objetivos) al invocar `delegate_task`.
- **Ejecución paralela:** Es posible ejecutar hasta 3 subagentes de forma concurrente, optimizando tareas de investigación, depuración o compilación.

## Ejemplo de uso (Parallel Batch)
Para delegar múltiples investigaciones paralelas:
```json
    {
      "tasks": [
        {"goal": "Investigar API de modelos de lenguaje", "context": "Documentación v1"},
        {"goal": "Verificar tests de integración", "context": "Repositorio actual"}
      ]
    }
```

Esta arquitectura permite que Hermes Agent escale sus capacidades de resolución de problemas sin contaminar el contexto principal.
